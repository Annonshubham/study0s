### **STEP 3: Enable GitHub Pages** (1 minute)

1. Go to your GitHub repo
2. Click **Settings** (top right)
3. Click **Pages** (left sidebar)
4. Under "Build and deployment":
   - Source: Select **Deploy from a branch**
   - Branch: Select **main**
   - Folder: Select **/ (root)**
5. Click **Save**
